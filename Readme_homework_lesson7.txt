Проект к 1-му и 2-му уроку оказался сложноват, поэтому для ДЗ использованы другие источники.

1. Найти и указать в проекте Front Controller и расписать классы, которые с ним взаимодействуют.
Файл - MainController.php из собственного проекта. MainController (= Front Controller) взаимодействует с классами ControlController и InitializeGPIO.

2. Найти в проекте паттерн Registry и объяснить, почему он был применён.
Файл - Registry.php   -> Честно "дернут" отсюда:  https://snipp.ru/php/registry  . В этом примере очень хорошее пояснение, как это работает. 

3. Добавить во все классы Repository использование паттерна Identity Map вместо постоянного генерирования сущностей.
Файл - Identitymap.php  -> За основу взят пример из методички, совмещен с ДЗ к уроку 4 и проверен на реальной БД.
